				캐치 마인드 프로그램( 졸업 작품 )

-  제작 인원 : 4 명

-  담당 : 프로그램 제작
 
-  설정 배경 : 2년동안 배운내용 복습
               이해를 하지 않고 또는 잘못이해한 부분에 대한 공부
               새로운 객체에 대한 공부

-  개발 목표 : 디자인을 제외한 넷마블사의 캐치마인드 프로그램의
              몇가지 기능을 구현

-  개발 내용 : MYSQL과 연동된 서버, swing을 이용한 gui 클라이언트

              * 서버
		 - JDBC를 이용한 mysql 연동
                 - 유저(스레드) 정보, 방 정보 객체화
		 - 브로드캐스팅 기능
                 - 접속중인 유저와 생성된 방에 대한 정보를 담을 내부객체 구현(유저[HashMap],방[Vector])
                      * 유저 객체
                         - 클라이언트로부터 받은 요청 처리(로그인,회원가입,방생성,입장,채팅,좌표전송 등)
                      * 방 객체
			 - 방에 접속한 유저정보 저장 (Vector)
			 - 브로드캐스팅 기능

	      * 클라이언트
		  - 5개의 프레임으로 구성 (로그인,회원가입,방목록,방생성,게임창)
		      * 게임 프레임
			 - 자유곡선 객체구현 및 자유곡선 정보를 저장할 ArrayList객체 생성
		  - 버튼 액션 및 마우스 액션에 대한 처리와 서버와 연결될 객체 구현(cilent)
                      * cilent 객체
			 - 소캣 생성 및 데이터를 보낼 스트림객체 생성
                         - 정보를 받을 내부객체 구현(receiver)
			 - 마우스 및 버튼액션에 대한 정보를 서버에 요청
